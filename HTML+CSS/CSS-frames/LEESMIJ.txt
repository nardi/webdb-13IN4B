In deze map laat ik drie techniekjes zien, waarmee je het effect van frames kunt 
bereiken zonder <frameset> en zonder <frame>

1. "CSS-frames.html"
Dit laat zien hoe je met CSS een pagina verdeelt in delen (DIV-elementen) die 
onafhankelijk van elkaar kunnen scrollen. Je kunt met CSS geen hyperlinks maken 
die deze 'frames' 'target'-en

2. "CSS-javascript-frames.html"
Met javascript kun je de inhoud van de DIV-s laten veranderen, als iemand op een 
'link' klikt (de links zijn p-elementen met een "onClick" attribuut). Je kunt 
geen files openen binnen de DIV's, dus echte navigatie is niet mogelijk.

3. "CSS-php-frames.html"
Met hulp van de PHP "include" opdracht kun je zorgen dat binnen de DIV de inhoud 
van een andere file verschijnt: echte navigatie met 'frames'zonder <frame>!  

4. CSS-ajax-frames
In deze map vind je css-ajax.php
De AJAX techniek wordt in week 3 uitgelegd, tegelijk met javascript.
